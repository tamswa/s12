# وظائف السعودية - Saudi Jobs Landing Page

صفحة هبوط للتوظيف في المملكة العربية السعودية للمواطنين والمقيمين.

## المميزات

- صفحة هبوط احترافية باللغة العربية (RTL)
- 9 وظائف متاحة (عمال نظافة، مندوب مبيعات، عمال تغليف، سائقين، وغيرها)
- نموذج تقديم يطلب: الاسم الثلاثي، المهنة، الجنسية، السن
- تحويل مباشر لرابط العرض بعد إرسال النموذج
- تصميم متجاوب يعمل على جميع الأجهزة (حواسيب وهواتف)
- حفظ البيانات في Supabase
- حركات وانتقالات سلسة

## التقنيات المستخدمة

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (قاعدة البيانات)
- Lucide React (الأيقونات)

## التشغيل

```bash
npm install
npm run dev
```

## البناء
